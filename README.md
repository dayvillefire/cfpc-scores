# CFPC SCORES

Retrieves CT Commission on Fire Prevention and Control test scores based on test date (MM/DD/YYYY) and last four digits of SSN.

# SYNTAX

```
Usage of ./cfpc-scores:
  -date string
    	Date of test (MM/DD/YYYY)
  -debug
    	Debugging enabled
  -dummy-data string
    	File with test data to parse
  -ssn string
    	Last 4 of SSN

```
